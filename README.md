A collection of **freely usable media assets** — images, logos, audio, and other creative resources — intended for use in projects, demos, websites, apps, videos, and experiments.

This repository exists to **remove friction**.  
No attribution requirements. No usage restrictions. No questions asked.

This repository may include (but is not limited to):
